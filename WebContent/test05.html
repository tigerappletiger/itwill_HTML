html5에서 추가된 input type의 속성값들

html5에서 모바일 환경에 알맞는 폼요소들이 추가됨. input type 값에 따라 형태,종류가 달라짐

input type="time" 사용자의 지역을 기준으로 시간을 지정

input type="date" 사용자의 지역을 기준으로 날짜와 시간을 지정

input type="datetime-local" 사용자의 지역을 기준으로

input type="week"

input type="month"

input type="number"

시간,날짜 숫자 등의 폼요소에는
min 속성으로 입력값의 최소치를,
max 속성으로 입력값의 최대치를,
step 속성으로 입력값의 간격을 지정할수있다.

  1. 개(min="0")
  2. 주문수량(max="100")
  3. 개(step="12")

input type="range" 범위내의 값을 지정

input type="tel" 연락처 모바일 브라우저에서 가상키패드가 숫자 입력 형식으로 바뀔수있다.

input type="email" 모바일 브라우저에서 가상키패드가 @등을 지원하게 바뀔수있다.

input type="url" 모바일 브라우저에서 알맞은 형식의 값인지 체크해줌 https://등을 지원하게 바뀔수있다..

input type="color"
