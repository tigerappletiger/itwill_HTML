form

form은 사용자와 서버를 연결하는 페이지 부분을 구성한다.
사용자가 정보를 서버에 전달할 때 이용하는 다양한 형태의 폼요소들로 이루어진다.
action 속성으로 정보를 전달 받을 서버측 파일을 지정하고
method 속성으로 정보 전달 방식을 지정한다.(일반적으로 속성값은 post를 지정)



폼요소들

input type="text" 한 줄 글상자(주관식)

input상자에 대한 설명문구를 label태그로 표현한다.
그리고 label 태그에는 for속성의 값을,
input 태그에는 id속성의 값을 서로 일치 시켜주면 서로 연결이 된다.


input상자의 속성

  • size="n" 상자의 길이
  • maxlength="n" 입력데이터의 최대 길이
  • minlength="n" 입력데이터의 최소 길이

input type="password" 암호 글상자(주관식)


textarea ... /textarea 큰 글상자(주관식)

rows="n" 속성으로 상자의 세로줄을,
cols="n" 속성으로 가로 길이(글자수)를 지정한다.

input type="radio" 단일 선택용 라디오버튼 (객관식)

단일 선택이 되게 하려면 같은 질문하의 라디오버튼 항목들끼리는 name 속성의 값을 서로 일치 시켜준다.
초기값으로 선택되어질 항목에는 checked="checked"(checked)속성을 붙여준다.

input type="submit/reset/button" 전송/취소/일반 버튼

value속성의 값으로 버튼의 눈에 보이는 글자를 지정할 수 있다.

input type="submit" 으로 전송용 버튼 생성

input type="reset" 으로 취소용 버튼 생성

input type="button" 으로 일반 버튼 생성

input type="image" 전송버튼 역할의 이미지

src속성으로 이미지의 경로를 지정함.
width속성으로 이미지의 가로 크기를,
height속성으로 이미지의 세로 크기를 조절할 수 있다.
